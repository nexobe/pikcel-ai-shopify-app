## Fastest Start

```typescript
// 1. Add to .env
PIKCEL_API_URL=https://api.pikcel.ai
PIKCEL_API_KEY=your_key_here

// 2. Import
import { getPikcelAIClient } from '~/services';

// 3. Use
const client = getPikcelAIClient();
const models = await client.getAIModels();
```
